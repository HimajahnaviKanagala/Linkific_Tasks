## ⚙️ Setup & Installation

1. Clone the repository:

```bash id="r3x2k8"
git clone <your-repo-link>
cd <project-folder>
```

2. Install dependencies:

```bash id="f4p0ns"
npm install
```

3. Install Bootstrap:

```bash id="x1z9c2"
npm install bootstrap react-bootstrap
```

4. Import Bootstrap in `main.jsx`:

```js id="u6l8qm"
import 'bootstrap/dist/css/bootstrap.min.css';
```

5. Run the app:

```bash id="q8n5yv"
npm run dev
```

## 🧠 Key Concepts Learned

### 🔹 Bootstrap Grid System

Used responsive layout with:

* `container`, `row`, `col`
* Breakpoints: `sm`, `md`, `lg`, `xl`

Example:

```html id="o9w2mz"
<div class="row">
  <div class="col-md-4">Column</div>
</div>
```

---

### 🔹 React Bootstrap

Used Bootstrap components as React components instead of plain classes.

Example:

```jsx id="p7t1d4"
<Button variant="primary">Click Me</Button>
```

## ⚔️ Bootstrap vs Tailwind CSS

| Feature       | Bootstrap            | Tailwind              |
| ------------- | -------------------- | --------------------- |
| Approach      | Component-based      | Utility-first         |
| Customization | Limited              | Highly customizable   |
| Speed         | Faster for beginners | Faster after learning |
| Design        | Predefined styles    | Fully custom designs  |
| Flexibility   | Moderate             | High                  |

---

## 🎯 When to Use Bootstrap

* Quick prototyping
* Prebuilt UI components needed
* Beginner-friendly projects

---

## 🎯 When to Use Tailwind

* Custom UI design
* Modern applications
* Full design control required

---

## 💡 Key Takeaways

* Bootstrap speeds up development with ready-made components
* React Bootstrap improves integration with React
* Grid system is essential for responsive design
* Component libraries reduce development time
