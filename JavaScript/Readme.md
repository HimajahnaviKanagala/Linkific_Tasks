# JavaScript Fundamentals Projects

This repository contains beginner projects built while learning **JavaScript fundamentals**, **ES6 features**, **DOM manipulation**, and **asynchronous programming**.

The goal of these projects is to practice core JavaScript concepts and build small interactive web applications.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6+)
* DOM Manipulation
* Fetch API
* Async / Await

---

## Projects

### 1. Todo List Application

A simple interactive Todo List where users can manage tasks.

**Features**

* Add new tasks
* Delete tasks
* Filter/search tasks

**Concepts Practiced**

* DOM manipulation
* Event listeners
* Array handling
* Dynamic element creation

---

### 2. Dynamic Form Validation

A registration form that validates user input using JavaScript.

**Features**

* Name validation
* Email format validation
* Password length validation
* Error message display

**Concepts Practiced**

* Form events
* Conditional logic
* Regular expressions
* DOM updates

---

### 3. Interactive Quiz Application

A quiz application that displays questions and calculates the user's score.

**Features**

* Multiple-choice questions
* Answer selection
* Score calculation
* Quiz restart functionality

**Concepts Practiced**

* Arrays and objects
* Event handling
* DOM manipulation
* JavaScript logic

---

### 4. Weather Application

A weather app that fetches real-time weather data from an API.

**Features**

* Search weather by city
* Display temperature
* Show weather condition

**Concepts Practiced**

* Fetch API
* Async / Await
* API integration
* Error handling
